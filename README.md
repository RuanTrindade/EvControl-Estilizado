## 📌 EvControl – Sistema de Gerenciamento de Reservas

Aplicação web para controlar reservas com calendário interativo, cadastro de clientes, valores e observações.

---

## ✅ Tecnologias Utilizadas
- React
- Axios
- React-Calendar
- CSS Responsivo
